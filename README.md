# Digital Agronomist 180-Day Curriculum

This repository documents my 180-day journey to build practical competence in digital agronomy, climate-smart agriculture, SQL, PostgreSQL, Python, geospatial analysis, machine learning basics, and dashboard development.

## Project Goal

The goal of this project is to build a reproducible digital agronomy portfolio that demonstrates how agricultural data can be structured, cleaned, analysed, mapped, modelled, and communicated for climate-smart decision support.

## Core Skills

- SQL and PostgreSQL
- Python programming
- Pandas data analysis
- Real agricultural data cleaning
- PostGIS and GeoPandas
- Remote sensing concepts
- Basic machine learning
- Streamlit dashboard development
- Reproducible research
- Scientific communication
